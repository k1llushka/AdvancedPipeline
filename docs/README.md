# Advanced Calculator

Продвинутый калькулятор с математическими и статистическими функциями.

## Функции

### Базовые операции
- Сложение
- Вычитание  
- Умножение
- Деление

### Расширенные операции
- Возведение в степень
- Факториал
- Проверка на простое число
- Статистические расчеты

## Использование

Запустите программу и увидите демонстрацию всех функций.
